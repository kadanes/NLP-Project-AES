Contains the 2 datasets:
1. Blog Dataset labeled with age and gender (https://www.kaggle.com/tomlisankie/blog-posts-labeled-with-age-and-gender): Used for
training the 2 models. Present in train.json
2. Generalization dataset for gender model: Blog Author Gender Classification Dataset (www.cs.uic.edu/~liub/FBS/blog-gender-dataset.rar):
Used to check how the gender model is performing. Present in blog-gender-dataset.csv
